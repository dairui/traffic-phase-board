【v0.1实现说明】

每块相位板有一个ID，取值为1~4
每块相位板最多可以控制4组灯的状态输出，每组灯是由红、黄、绿三个灯组成的灯组。现编号如下：

第一组           第二组            第三组             第四组
红灯Rx1           红灯Rx2          红灯Rx3            红灯Rx4
黄灯Yx1           黄灯Yx2          黄灯Yx3            黄灯Yx4
绿灯Gx1           绿灯Gx2          绿灯Gx3            绿灯Gx4

其中，x取值为1~4，代表本相位板的ID。

机动车通道使能位图为：
   (高四位)      (第四组灯是否为机动车道灯组)  (第三组灯是否为机动车道灯组)  (第二组灯是否为机动车道灯组)   (第一组灯是否为机动车道灯组)
0   0   0   0                Vx4                           Vx3                            Vx2                                Vx1

人行通道使能位图为：
   (高四位)      (第四组灯是否为行人通道灯组)  (第三组灯是否为行人通道灯组)  (第二组灯是否为行人通道灯组)   (第一组灯是否为行人通道灯组)
0   0   0   0                Wx4                           Wx3                            Wx2                                Wx1

主控板下发的数据格式应符合以下说明：

第一条数据：
0xC1  0x01  0xB1  0x00  0xXX  0b(R24 R23 R22 R21 R14 R13 R12 R11) 0b(Y24 Y23 Y22 Y21 Y14 Y13 Y12 Y11) 0b(G24 G23 G22 G21 G14 G13 G12 G11)

第二条数据：
0xC1  0x02  0xB1  0x00  0xXX  0b(R44 R43 R42 R41 R34 R33 R32 R31) 0b(Y44 Y43 Y42 Y41 Y34 Y33 Y32 Y31) 0b(G44 G43 G42 G41 G34 G33 G32 G31)

其中，0xXX为任意值，相位板不做解析。

第三条数据：
0xC1  0x04  0xB1  0x00  0b(V24 V23 V22 V21 V14 V13 V12 V11)   0b(V44 V43 V42 V41 V34 V33 V32 V31)   0b(W24 W23 W22 W21 W14 W13 W12 W11)   0b(W44 W43 W42 W41 W34 W33 W32 W31)
